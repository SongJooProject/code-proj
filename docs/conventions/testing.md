# 코딩 규칙: 테스트

## 자체 검증 스크립트
- 각 프로젝트는 결과물 검증용 스크립트를 별도로 작성
- 스크립트는 `test_*.py` 또는 `validate_*.py` 형식
- 작업 완료 후 반드시 실행하여 결과물 품질 확인

## 테스트 명령어
```bash
# 뻥뚜리 문제 생성
python "D:\code proj\word proj\test_quiz.py"
```

## 검증 항목
1. **문제지 검증**
   - 빈칸 형식 확인 (`▁N▁`)
   - 문단당 최대 2개 빈칸
   - 빈칸 번호 유효성

2. **답안지 검증**
   - 섹션별 번호 1부터 시작
   - 번호 순서 정상
   - 답안 존재 여부

3. **일관성 검증**
   - 문제지-답안 매칭
   - 단어 존재 여부
