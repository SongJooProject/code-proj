# ADR 002: 섹션별 번호 restart

## 날짜
2026-06-20

## 상태
수용

## 결정
빈칸 번호를 문단별이 아닌 **섹션별**로 restart하기로 결정하였다.
상위 섹션을 포함하여 고유한 섹션 이름을 생성한다.
표는 별도 섹션으로 분리한다.
**섹션 내 중복 단어를 제거**한다.

## 고려된 대안
1. 전체 연속 번호 — 번호가 너무 커짐
2. 문단별 restart — 섹션 구분이 불분명
3. 섹션별 restart — 섹션마다 1번부터 시작

## 근거
- 학생이 특정 섹션에서 빈칸을 풀 때 번호가 작으면 접근성 향상
- 답안지에서 【섹션 제목】 + 번호로 구성하면 찾기 쉬움
- 같은 이름의 섹션이 여러 번出现할 수 있으므로 상위 섹션 포함하여 고유하게 만듦
- 표는 본문과 구분하여 별도로 관리하는 것이 효율적
- 섹션 내 같은 단어가 여러 번出现하면 혼동되므로 중복 제거

## 결과
- 답안지가 섹션별로 깔끔하게 구분됨
- 상위 섹션 포함: `I. 공무원의 직무상 불법행위 > 1. 관할`
- 표 섹션: `표 1`, `표 2`, ...
- 섹션 내 중복 단어 제거
- 학생이 특정 영역만 집중적으로 풀 때 유용
