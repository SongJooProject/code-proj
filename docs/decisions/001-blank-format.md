# ADR 001: 빈칸 형식 채택

## 날짜
2026-06-20

## 상태
수용

## 결정
행정법 빈칸 채우기 문제의 빈칸 형식으로 `▁1▁` (줄 + 번호 + 줄)을 채택하였다.

## 고려된 대안
1. `________` (줄만) — 구분 불가
2. `(1)` — 원본 문서에 이미 사용 중인 번호와 충돌
3. `①②③` — 최대 20개까지만 가능하고, 원본과 혼동
4. `▁1▁` — 구분 가능, 번호 표시, 충돌 없음

## 근거
- 원본 문서에 ①② 형식이 이미 사용되어 구분 불가
- 학생이 빈칸 위치와 순서를 쉽게 파악할 수 있어야 함
- 넘버링이 20개를 초과할 수 있으므로 원형 문자 사용 불가

## 결과
- 빈칸 위치가 명확하게 표시됨
- 답안에서 번호로 빈칸을 찾기 쉬움
