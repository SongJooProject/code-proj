# ADR 003: 답안 형식 변경

## 날짜
2026-06-20

## 상태
수용

## 결정
답안 형식을 `문단번호-빈칸번호. 단어`에서 `N번. 단어`로 변경하였다.

## 고려된 대안
1. `7-1. 공무원` (문단번호-빈칸번호) — 섹션별 restart와 충돌
2. `1번. 공무원` (섹션별 번호) — 단순하고 명확

## 근거
- 섹션별 번호 restart를 적용하면서 문단번호가 불필요해짐
- 학생이 답안을 볼 때 섹션 제목과 번호만 알면 충분함
- 형식이 단순하면 오답 확인이 쉬움

## 결과
- 답안지가 깔끔하게 정리됨
- 【섹션 제목】 아래에 번호가 1부터 순서대로 표시
