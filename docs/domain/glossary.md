# 용어 사전

## 뻥뚜리
- 행정법 빈칸 채우기 문제를 자동 생성하는 에이전트
- 원본 docx 파일을 읽어서 빈칸 문제와 답안을 생성

## 빈칸 형식
- `▁1▁`, `▁2▁` 형식
- 줄 + 번호 + 줄로 구성
- 문단당 최대 2개

## 섹션
- 문제지의 제목이나 분류 단위
- 【섹션 제목】으로 표시
- 섹션별로 번호가 restart됨

## 답안 형식
- `N번. 단어` 형식
- 섹션별로 그룹핑하여 표시
- 예: 【헌법적 근거】 1번. 손해, 2번. 손해

## 핵심 단어
- `law_terms_db.json`에 저장된 핵심 용어
- 국가배상법, 지방자치단체법, 토지보상법 등
- 문제 생성 시 이 단어들을 빈칸으로 만듦
