## 뻥뚜리 에이전트 (행정법 빈칸 문제 생성)

### 작업 규칙
1. **세션 시작 시**: `뻥뚜리_명세서.md` 반드시 읽기
2. **작업 중**: 명세서의 규칙 준수
3. **작업 완료 후**: 명세서와 progress.md 업데이트

### 핵심 파일
- `畅通리_명세서.md` — 요구사항 명세서
- `law_terms_db.json` — 핵심 단어 DB
- `create_quiz_db.py` — 문제 생성 스크립트
- `test_quiz.py` — 결과물 검증 스크립트

### 작업 완료 시 검증
```bash
python "D:\code proj\word proj\test_quiz.py"
```

### 빈칸 규칙
- 문단당 최대 2개 빈칸
- **섹션 내 중복 단어 제거** (같은 섹션에서 같은 단어가 여러 번出现해도 한번만)
- **섹션별 번호 restart** (섹션 변경 시 1번부터)
- **상위 섹션 포함**하여 고유한 섹션 이름 생성 (예: `I. 섹션 > 1. 하위섹션`)
- **표는 별도 섹션** (표 1, 표 2, ...)
- 빈칸 형식: `▁1▁`, `▁2▁`
- 답안 형식: `N번. 단어`

### 답안 형식
```
【헌법적 근거】
  1번. 손해
  2번. 공무원
【I. 공무원의 직무상 불법행위 > 1. 관할】
  1번. 국가
【표 1】
  1번. 공무원
  2번. 권력작용
```
