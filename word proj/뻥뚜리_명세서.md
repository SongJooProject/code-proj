# 뻥뚜리 - 행정법 빈칸 채우기 문제 생성 에이전트

## 목적
행정법 관련 docx 파일을 읽어서 빈칸 채우기 문제/답안을 생성하는 에이전트

## 요구사항 명세서

### 1. 입력
- `word proj/` 폴더 내 docx 파일 5개

### 2. 출력
- 파일명 형식: `{원본파일명}_문제.docx`, `{원본파일명}_답안.docx`

### 3. 핵심 규칙 (반드시 지킬 것)

#### ⭐ 양식 유지 (가장 중요)
- **원본 docx 파일을 복사한 후 단어만 변경**
- 새로 문서를 만들지 않음
- 표(table), 폰트, 크기, 정렬, 간격 등 100% 유지

#### ⭐ 빈칸 규칙
- **문단당 최대 2개 빈칸**
- **문단별 번호 restart** (각 문단에서 1번부터 시작)
- 빈칸 형식: `▁1▁`, `▁2▁` (줄 + 번호 + 줄)
- DB에 있는 핵심 단어만 문제로 출제

#### ⭐ 답안 규칙
- **섹션별 그룹핑** (【섹션 제목】으로 구분)
- 형식: `문단번호-빈칸번호. 단어`
- 예시:
  ```
  【헌법적 근거】
    2-1. 손해
    3-1. 손해
  【요건】
    8-1. 공무원
    9-1. 공무위탁
  ```

### 4. 작업 절차
1. 원본 docx 파일 복사 (shutil.copy2)
2. 복사된 파일을 python-docx로 열기
3. 각 문단(paragraph)과 표(table)을 순회
4. 핵심 단어를 `▁1▁` 형식으로 변경
5. 저장

### 5. 빈칸 대상 단어 선정 기준
- law_terms_db.json에 등록된 핵심 용어만 사용
- 인터넷 검색으로 법령 관련 핵심 단어 선정 후 DB화
