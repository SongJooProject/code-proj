# 뻥뚜리 - 행정법 빈칸 채우기 문제 생성 에이전트

## 목적
행정법 관련 docx 파일을 읽어서 빈칸 채우기 문제/답안을 생성하는 에이전트

## 요구사항 명세서

### 1. 입력
- `word proj/` 폴더 내 docx 파일 5개

### 2. 출력
- 파일명 형식: `{원본파일명}_문제.docx`, `{원본파일명}_답안.docx`

### 3. 핵심 규칙 (반드시 지킬 것)

#### ⭐ 양식 유지 (가장 중요)
- **원본 docx 파일을 복사한 후 단어만 변경**
- 새로 문서를 만들지 않음
- 표(table), 폰트, 크기, 정렬, 간격 등 100% 유지

#### ⭐ 빈칸 규칙
- **각 문단에서 최소 5개의 빈칸** 만들기
- 원문의 핵심 법조문, 판례 개념에서 단어 선정
- 빈칸 단어는 완전히 일치하는 것만 변경 (부분 매칭 금지)

#### ⭐ 답안 규칙
- **문단 제목 + 빈칸 순서대로 번호 매김**
- 형식:
  ```
  문단 제목: I. 공무원의 직무상 불법행위로 인한 배상책임
  1. 불법행위
  2. 손해
  3. 국민
  4. 법률
  5. 국가
  ```

### 4. 작업 절차
1. 원본 docx 파일 복사 (shutil.copy2)
2. 복사된 파일을 python-docx로 열기
3. 각 문단(paragraph)과 표(table)을 순회
4. 핵심 단어를 ________로 변경
5. 저장

### 5. 빈칸 대상 단어 선정 기준
- 헌법, 법률의 핵심 조문에 등장하는 용어
- 판례에서 인용하는 핵심 법리
- 개념 정의에 사용되는 핵심 단어
- 요건, 효과, 절차에서 빠져서는 안 되는 단어

### 6. 현재 진행 상황
- [x] 국가배상, 손실보상 원본 파일 확인
- [ ] 국가배상, 손실보상 빈칸 문제 생성 (문단별 5개 이상)
- [ ] 나머지 4개 파일 문제 생성

## 메모
- python-docx의 `shutil.copy2`로 원본 복사 후 단어 변경이 가장 간단
- 새로 문서를 만들면 표, 스타일 등이 모두 사라짐
